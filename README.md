# FP Selection — Galeria de Carros

![image alt](https://github.com/TheusCampos/landingpage-site-car-animation/blob/4ed63b97ee0a26167afbe63f661736cf4517cf87/img/Site-car.png)

Um projeto front-end feito para treinar animações, composição de layout e interações com JavaScript — simulando uma vitrine de carros premium.

O ponto de partida foi um vídeo do YouTube (créditos abaixo). A partir dessa inspiração, o projeto foi evoluindo com novas seções, refinando o visual e ajustando o layout para ficar responsivo em diferentes tamanhos de tela.

## O que você encontra aqui

- **Slider/galeria de carros** na seção Home, com setas e indicadores
- **Seção Destaque** com chamada para ação e especificações do carro
- **Seção “Fale Conosco”** com layout moderno e links sociais
- **Responsividade** (mobile/desktop) e otimizações simples como `loading="lazy"` em imagens

## Tecnologias

- HTML5
- CSS3 (animações e layout responsivo)
- JavaScript (controle do slider)

## Estrutura do projeto

- [index.html](./index.html): estrutura das seções (Home, Destaque, Contato) e navegação
- [assets/css/style.css](./assets/css/style.css): estilos, animações e responsividade
- [assets/js/script.js](./assets/js/script.js): lógica do slider (próximo/anterior, item ativo e indicadores)
- [img](./img): imagens usadas no layout

## Como executar

1. Abra o arquivo [index.html](./index.html) no navegador, ou use uma extensão como “Live Server”.
2. Navegue pelo slider usando as setas e teste a página em tamanhos diferentes (desktop e mobile).
